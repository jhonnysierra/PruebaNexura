# PruebaNexura
 Prueba técnica para desarrollador PHP empresa Nexura
La aplicación se realizó en un ambiente local usando wampserver 3.1.7, version de PHP 7.3.1, versión apache 2.4.37.
Se usó para escribir el código el modelo MVC sin hacer uso de frameworks.
La aplicación se desarrollo hasta el punto que puede crear un empleado en el sistema. No se realizaron validacion y el resto de requisitos debido a que el tiempo no me alcanzó.
